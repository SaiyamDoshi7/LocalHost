React LocalBox Miner Project Project Description

This project is a simple React application that allows users to manage records using a form. Users can add, edit, delete, and view records. The data is stored in the browser using LocalStorage so that the records remain saved even after refreshing the page.

This project demonstrates basic CRUD operations in React.

Modules Explanation

Add Record Module
This module allows users to enter details in the form and save them. The data is stored in LocalStorage.

Record List Module
All saved records are displayed in a table format. Users can easily view all records.

Edit Module
Users can edit existing records. When the Edit button is clicked, the form is filled with the existing data so the user can update it.

Delete Module
Users can delete records from the list. A confirmation message appears before deleting the record.

LocalStorage Integration
All records are saved in LocalStorage so the data is not lost when the page is refreshed.

Steps to Run the Project

Download or clone the repository

Open the project folder in terminal

Install dependencies

npm install

Start the development server

npm run dev

Open the browser and visit the shown local URL.
